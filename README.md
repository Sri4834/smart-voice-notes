# 🎙️ Smart Voice Notes

An intuitive Android application designed for hands-free note-taking. It uses advanced voice capturing to convert speech to text in real-time, allowing users to save and share their thoughts instantly.

## ✨ Key Features
*   **Voice-to-Text**: Real-time conversion using Google Speech-to-Text API.
*   **Multi-Format Export**: Automatically generate and save notes as `.txt` and `.pdf` files.
*   **Organized Storage**: Dedicated internal storage for easy retrieval of saved notes.
*   **Instant Sharing**: Share your generated text or PDF documents directly via WhatsApp, Email, or other platforms.

## 🛠️ Tech Stack
*   **Platform**: Android
*   **Language**: Java / Kotlin
*   **UI**: Material Design (XML)
*   **Libraries**: iText (for PDF generation), Android Speech Recognition.

## 🚀 How to Use
1.  **Clone the Repo**: `git clone https://github.com`
2.  **Open in Android Studio**: Wait for Gradle to sync.
3.  **Permissions**: Ensure Microphone and Storage permissions are granted on your device.
4.  **Capture**: Press the microphone icon and start speaking!
